# Financial Sales & Profit Analysis — SQL Project

## Overview
SQL analysis of global financial performance data examining sales, 
profit margins, and monthly trends across countries and products.

## Dataset
Microsoft Financial Sample — contains sales data across 6 countries, 
5 products and 18 months of transactions.

## Queries
1. **Sales & Profit by Country** — identifies top performing markets 
   and profit margin percentage per country
2. **Product Performance Analysis** — ranks products by profitability 
   and units sold
3. **Monthly Profit Trend** — tracks sales and profit movement over 
   time to identify seasonal patterns

## Skills Demonstrated
- SQL aggregation and grouping
- Data type casting and string cleaning
- Financial KPI calculation (profit margin %)
- Trend analysis

## Tools Used
- SQLite / DB Browser for SQLite
- Microsoft Excel (data source)
